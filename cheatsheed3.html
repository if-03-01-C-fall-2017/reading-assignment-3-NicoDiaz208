HTML for Lists

  • Unordered HTML Lists
    • ul defines an unordered list
  • Ordered HTML Lists
    1. ol defines an ordered list ("1.2.3.")
    HTML Description Lists
    -A Description list is a list of Terms, with a description of them.
  • List Items
    • List items can contain new list, and other HTML elements, like images and links, etc.

Tables

Tables are defined with the table tag Each table row is defined with the tr tag.
A table header is defined with the th tag. By default , table headings are bold and centered.
A table data/cell is defined with the td tag.
